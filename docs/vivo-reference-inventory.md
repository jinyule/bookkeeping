# vivo 钱包记账参考采集

采集设备：vivo V2307A（PD2307），Android 16，vivo 钱包 `com.vivo.wallet` 版本 `5.3.7.0`。

隐私处理：vivo 钱包原始截图、UI XML 和本应用验证截图可能包含真实商户、金额、账户等信息，已通过 `.gitignore` 忽略：

- `reference/`

## 已采集流程

- 钱包入口：首页 “记账” 入口。
- 记账首页：顶部 “记账日报 / 搜索 / 编辑”，汇总卡片，预算提示，按日期分组的账单列表，底部五栏导航。
- 记一笔：支出/收入切换，金额、日期、时间、分类、商户、账户、备注和数字键盘。
- 收入分类：收转账、薪资、理财、收红包、借入、收款等。
- 报表：周报/月报/年报/自定义，收支汇总，趋势和分类构成。
- 设置：自动记账设置、记账通知、分类管理、导入数据、帮助反馈等。
- 自动记账来源：短信通知、支付宝、云闪付、京东、美团、抖音、拼多多、钱包/vivo Pay、账单导入。
- 导入数据：微信/支付宝、Excel/CSV 等导入说明。
- 分类管理：支出/收入分类网格。

## 取舍

- 保留 vivo 记账的核心信息架构：五栏导航、汇总卡、分组账单、底部记账面板、报表、自动记账来源、设置和导入。
- 排除营销、广告、贷款、应用商店推荐、主题卡片等非记账功能。
- 用户补充要求优先：本 App 的自动记账明确支持微信支付，即使 vivo 钱包参考页显示微信自动记账内测结束。
